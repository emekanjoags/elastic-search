## Install
- Clone the repository:
```
git clone git@github.com:emekanjoags/elastic-search.git
```

- Check into the folder:
```
cd breaking-bad
```
- Build Application
```docker-compose build```

- Run Application
```docker-compose up```

- To Stop the server
```docker-compose down```
